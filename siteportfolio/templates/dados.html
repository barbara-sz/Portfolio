PowerBI

O PowerBI acaba sendo muito prático para o todo o processo de análise, já que toda a limpeza dos dados e a construção da visualização é feita em um só lugar.

Esta base foi extraída do kaggle e se trata de informações de lançamento e venda de jogos para diferentes consoles de videogames.

Os dados são bem básicos e intuitivos, tornando a construção dos elementos mais fácil. As informações estão em milhões de dólares e se baseiam nos jogos, em regiões, plataforma (console) e gênero.

Plotly

Apesar de o PowerBI ser muito útil, existe outras ferramentas que são excelentes para análise e visualização de dados, como a biblioteca dash do python.

O plotly traz muitas formas de explorar os dados, para várias finalidades. A própria biblioteca tem como objetivo a publicação online dos dashboards por meio de uma página html e segue a mesma lógica utilizada no Flask. Com alguns ajustes, é possível integrar as duas ferramentas gerando um conteúdo visual de qualidade.

Esta base de dados foi criada através do ChatGPT e é um exemplo típico que contém informações de vendas de uma empresa com quantidade de produtos vendidos, preço unitário, faturamento, categorias e localização.

Estes painéis são somente uma pincelada sobre análise de dados, mas escolhi esta forma por ser mais visual.

Apesar disso, a parte de tratamento e modelagem dos dados têm papel fundamental, sendo a base para que sua análise seja bem feita.
